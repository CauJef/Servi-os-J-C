1. Fluxo de Transação (Escrow)
Pagamento: 100% antecipado via Pix (Pagar.me) para a plataforma.
Garantia: Dinheiro retido até confirmação de ambas as partes.
Liberação: Confirmação do Prestador + Confirmação do Cliente (ou 24h de silêncio do cliente).
2. Gestão de Escopo e Ajustes
Adicionais: Funcionalidade para o prestador solicitar "Aditivo de Valor" via app.
Paiol de Provas: Chat interno deve permitir envio de fotos (antes/depois) para resolver disputas manuais.
3. Prevenção de Bypass (Anti-Pix por fora)
Taxa de Deslocamento: Valor fixo retido em caso de cancelamento após o aceite.
Bloqueio de Dados: Filtro de strings no chat (Regex para telefones/chaves Pix).
Fidelidade: Sistema de reputação (Rank) atrelado ao volume de transações pagas via app.
4. Logística Regional (Sem Google Maps)
Geolocalização: Uso de CEP + Fórmula de Haversine (linha reta) ou Tabela de Bairros Fixa.
Custo Zero: Integração com ViaCEP (grátis) para conversão de endereços.
5. Resolução de B.O. (MVP)
Disputa: Botão "Não Confirmo" que bloqueia o repasse e gera alerta para os fundadores (vocês) resolverem via suporte.
